Welcome to DoNotes

Let's set up your AI-powered productivity bot in about 10 minutes.

What you'll need:

  • Telegram account — to create a bot and interact with it
  • OpenAI account — for voice transcription & smart analysis
  • Google account — for Gmail, Calendar, and Sheets integration

What DoNotes does:

Send a voice note or text via Telegram. DoNotes transcribes it, extracts action items and calendar events, creates Google Calendar entries, logs to Sheets, and sends email digests — all automatically.

Step 3 of 8

Set Up Google Cloud

DoNotes needs access to Gmail, Google Calendar, and Google Sheets. Follow these steps in the Google Cloud Console:

1

Go to console.cloud.google.com and create a new project

Click the project dropdown at the top → "New Project" → name it "DoNotes" → Create
2

Enable these three APIs in APIs & Services → Library:

  • Gmail API
  • Google Calendar API
  • Google Sheets API
3

Configure the OAuth consent screen:

APIs & Services → OAuth consent screen → External → Fill in app name ("DoNotes") and your email → Add yourself as a test user
4

Create an OAuth 2.0 Client ID:

APIs & Services → Credentials → "+ Create Credentials" → "OAuth client ID" → Application type: Desktop app → Create
5

Click "Download JSON" on the popup to download your credentials file

You'll upload this file in the next step

Tip: If your OAuth consent screen is in "Testing" mode, tokens expire every 7 days. To avoid this, click "Publish App" on the consent screen — no review is needed for personal use.
